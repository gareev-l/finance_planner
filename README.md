# Финансовый планер

Модульное Python-приложение для учета личных финансов с асинхронной обработкой данных.

## Возможности

- Регистрация финансовых операций (доходы/расходы)
- Хранение данных в SQLite с асинхронным доступом
- Категоризация операций
- Фильтрация и сортировка
- Расчет баланса и агрегированных показателей
- Визуализация данных (графики, диаграммы)
- Экспорт/импорт CSV и JSON
- Web-интерфейс на Flask
- Unit-тесты

## Технологии

- Python 3.8+
- Flask (web-интерфейс)
- aiosqlite (асинхронная БД)
- pandas (анализ данных)
- matplotlib (визуализация)
- asyncio (асинхронное программирование)

## Установка
```bash
# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Установка зависимостей
pip install -r requirements.txt

# Создание директории данных
mkdir data
```

## Запуск
```bash
python main.py
```

Откройте браузер: http://127.0.0.1:5000

## Тестирование
```bash
python test_app.py
```

## Асинхронные операции

Проект использует asyncio и aiosqlite для:
- Асинхронного чтения/записи в БД
- Параллельной обработки пакетов операций
- Неблокирующего расчета статистики

## Примеры использования

### Добавление операции
1. Выберите тип (доход/расход)
2. Введите сумму
3. Укажите категорию
4. Выберите дату
5. Добавьте комментарий (опционально)

### Просмотр статистики
Переход в раздел "Статистика" для просмотра:
- Расходов по категориям
- Динамики доходов/расходов
- Топ категорий

### Экспорт данных
Выберите формат (CSV/JSON) для экспорта всех операций.
